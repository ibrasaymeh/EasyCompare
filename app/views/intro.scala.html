About
  • Easy Compare is a sensor based information retrieval system
  • This proof of concept is an example of what is possible if NFC tags are made ubiquitous (similar to barcodes)
  • Retail stores such as Best Buy, Walmart, and IKEA place paper-based information leaflets for pickup near various items for sale
  • Easy Compare revolves around the idea that stores have placed NFC tags next to each product (similar to how barcodes are presently used)
  • The Easy Compare app allows you to tap a tag to download and save information about a product on your mobile device
  • Allows you to compare products that are sold by competing stores
  • Stores can use NFC tags to distribute coupons exclusively to shoppers who have visited their store
Note: In order to preserve the integrity of the web application while keeping as much of the the original functionality present, the back end code for changing passwords and deleting tags and coupons has been disabled.
